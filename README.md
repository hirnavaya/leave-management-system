# Full Stack Leave Management System

A full-stack Leave Management System built using React, Node.js, Express, and Supabase (PostgreSQL).

## Features
- Employee management (create & view employees)
- Leave request submission
- Leave approval/rejection system
- Automatic leave day calculation
- Dashboard with summary statistics

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: Supabase (PostgreSQL)

## API Endpoints
- GET /employees
- POST /employees
- GET /leave
- POST /leave
- PATCH /leave/:id
- GET /dashboard

## How to Run

### Backend
cd server
npm install
npm run dev


### Frontend
cd client
npm install
npm run dev
